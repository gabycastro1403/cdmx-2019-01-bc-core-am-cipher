Bienvenidos al cifrado de cartas

Esta página esta creada para los papás que quieren mantener oculta su lista de compras para cualquier sorpresa que quieran darle a tus hijos. Puede ser:

  1. Lista de regalos de su cumpleaños.
  2. Lista de regalos para Dia de Reyes.
  3. Lista de compras para Navidad.

Puedes darle un sin fin de utilidades a esta página.

Lo único que tienes que hacer es :

  1. Ingresa el número de espacios que deseas que se recorra el alfabeto.
  2. El mensaje a codificar.

¡Listo tienes tu msj codificado!

Para decodificar tu mensaje solo tienes que:

  1. Ingresar el número de espacios con el que codificaste el msj.
  2. Ingresar el msj a decifrar.

¡Listo ya tienes tu msj descifrado!
